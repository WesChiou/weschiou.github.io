Content attributes
count Number The total pages. <shadow-pagination count="5"></shadow-pagination>
value Number The current page. <shadow-pagination count="5" value="3"></shadow-pagination>
circular boolean attribute Once the attribute is set, press the "<" button when you are on the page 1 to jump to the last page, and press the ">" button when you are on the last page to jump to the page 1. <shadow-pagination count="5" value="3" circular></shadow-pagination> Click "<", ">" to try it:
show-first-button boolean attribute Show the "⇤" button. Click the "⇤" button to go to page 1. <shadow-pagination count="5" value="3" show-first-button></shadow-pagination> Click "⇤" to try it:
show-last-button boolean attribute Show the "⇥" button. Click the "⇥" button to go to the last page. <shadow-pagination count="5" value="3" show-last-button></shadow-pagination> Click "⇥" to try it:
hide-previous-button boolean attribute Always hide the "<" button. <shadow-pagination count="5" hide-previous-button></shadow-pagination>
hide-next-button boolean attribute Always hide the ">" button. <shadow-pagination count="5" hide-next-button></shadow-pagination>
next-smart-mode enumerated attribute
    acceptable values:
  • hidden
  • disabled
 Hide (or disabled) the ">" button while at the last page.
  1. <shadow-pagination count="5" value="4" next-smart-mode="hidden"></shadow-pagination>
  2. <shadow-pagination count="5" value="4" next-smart-mode="disabled"></shadow-pagination>
    Click ">" to try it:
previous-smart-mode enumerated attribute
    acceptable values:
  • hidden
  • disabled
 Hide (or disabled) the "<" button while at page 1.
  1. <shadow-pagination count="5" value="4" previous-smart-mode="hidden"></shadow-pagination>
  2. <shadow-pagination count="5" value="4" previous-smart-mode="disabled"></shadow-pagination>
    Click "<" to try it:
ellipsis-style enumerated attribute
    acceptable values:
  • hidden
  • ellipsis (default)
  • no-ellipsis
 Omit those pages that far away from the current page, replace those pages into '...'(ellipsis), ''(hidden). Use no-ellipsis to not ellipsis.
You can control more details about ellipsis using attributes below.
  1. default(ellipsis): <shadow-pagination count="9" value="5"></shadow-pagination>
  2. hidden: <shadow-pagination count="9" value="5" ellipsis-style="hidden"></shadow-pagination>
  3. no-ellipsis: <shadow-pagination count="9" value="5" ellipsis-style="no-ellipsis"></shadow-pagination>
  2. hidden look like strange, can I hide the starting pages and the ending pages? Sure, we will discuss it later.
ellipsis-diameter non-negative numbers greater than zero
(1-2147483647)		 Decide how many pages to display around the current page.
  1. <shadow-pagination count="99" ellipsis-diameter="7"></shadow-pagination>
  2. <shadow-pagination count="99" value="50" ellipsis-diameter="7"></shadow-pagination>
  3. <shadow-pagination count="99" value="95" ellipsis-diameter="7"></shadow-pagination>
ellipsis-headsize non-negative numbers
(0-2147483647)		 Decide how many pages to display at head even it is far away from current page. <shadow-pagination count="9" value="9" ellipsis-headsize="2"></shadow-pagination>
ellipsis-tailsize non-negative numbers
(0-2147483647)		 Decide how many pages to display at end even it is far away from current page. <shadow-pagination count="9" ellipsis-tailsize="3"></shadow-pagination>
ellipsis-blursize non-negative numbers
(0-2147483647)		 When you get a satisfactory pagination like this: (that means you don't want to change the value of ellipsis-headsize, ellipsis-tailsize and ellipsis-diameter)You might think that 1 and 3 are so close, can I display 2 as well?
Yes, you can. When you think the ellipsis is unnecessary, use the "ellipsis-blursize" attribute to override the "ellipsis" related rules. When the number of omitted pages is below(or same as) the value of the "ellipsis-blursize" attribute, the ellipsis rule is not applied.		 <shadow-pagination count="99" value="5" ellipsis-blursize="1"></shadow-pagination>

pagesFormatFn Function Allow you set a function to format page number. shadow-pagination will run it before rendering each page. 

<shadow-pagination
  id="pages-format-fn-example"
  count="12
></shadow-pagination>

<script>
  const shadowPagination = document
    .querySelector('#pages-format-fn-example');
  if (shadowPagination) {
    shadowPagination.pagesFormatFn = (page) => {
      if (+page < 10) {
        return '0' + page;
      }
      return page;
    }
  }
</script>
Pages formated: !NOTE: DOM attributes only accept strings, can not set pagesFormatFn by content attribute.
